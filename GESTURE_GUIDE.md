# Gesture Recognition System - Advanced User Guide (V2)

This system now supports **50+ common words and phrases** in addition to the full alphabet and numbers.

## 🛡️ Accuracy & Stability Features
- **Temporal Smoothing**: The system now analyzes multiple frames (Consensus Engine) to ensure the gesture is correct before displaying it. This eliminates flickering.
- **Hand-Size Normalization**: The system automatically adjusts for how close you are to the camera.

## 🔠 Full Alphabet & Numbers
- **Letters**: A-Z (Standard ASL manual signs).
- **Numbers**: 0-9.

## ⌨️ Control Gestures
- **ENTER**: Closed Fist (Submits sentence for translation).
- **SPACE**: Index, Middle, and Ring fingers up.
- **BACKSPACE**: Thumb, Index, and Middle tips touching together.
